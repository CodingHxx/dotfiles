# Dotfiles

My personal Linux desktop configuration files and scripts.

## Included Configurations

* Hyprland
* Waybar
* Rofi
* Kitty
* Conky
* Spicetify
* Spotify Player
* Spotifyd
* WirePlumber
* XFCE Settings
* Zed Editor
* Custom scripts and themes

## Installation

Clone the repository:

```bash
git clone https://github.com/CodingHxx/dotfiles.git
cd dotfiles
```

Copy the required configuration files to your home directory:

```bash
cp -r .config ~/
```
